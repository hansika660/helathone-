#  Product Overview

Health One is a digital healthcare ecosystem that connects patients, doctors, diagnostics, pharmacies, and health records into one intelligent platform.

The platform aims to make healthcare:
- Accessible
- Affordable
- Connected
- Preventive

#  Installation Guide

## Clone Repository

```bash
git clone https://github.com/hansika660/healthone.git
```

## Open Project Folder

```bash
cd healthone
```

## Install Dependencies

```bash
npm install
```

## Run Project

```bash
npm start
```
